Preliminary example

But first let's talk about a simple example. You see we are given a first order differential equation that describes the wire diagram on the top left. Let's look at that wire diagram first: There are two Quantities - a Signal S and a Response R- that are correlated by those "wires" in between. Solid represent either the increase of the quantity they are pointing to or the decrease of the quantity which they are pointing away from. This would mean in a biological sense that protein R is synthesized through an arrow pointing to it and degraded by an arrow pointing away. Now there is also a second type of arrow, which is dashed. The dashed arrow means that one quantity influences a solid arrow in its magnitude. Again, speaking in a biological sense this means that one protein (the signal) influences the synthetization rate of the response protein. This information is combined in a short differential equation that takes the aggregation into account (blue) and the depletion (red). For your understanding: k0 is a base aggregation rate and the depletion always needs some R to be present in order to deplete it. Since the DE qualifies as an ODE we can solve it quite easily, which gives us the so called 'steady state' below. This depends only on the coefficients...and the Signal S. The plot to the right shows the calculated steady state solution and you can observe a linear dependency between the Signal and the steady state of the response. Keep in mind that these steady states are stable and as such the response will always tend towards the steady state for each signal, no matter what the current value of the response is. In this graph we can play around a little and change the values of some variables and as you can see, an increase in aggregation rate causes the response curve to be steeper while an increase of the depletion rate causes the opposite. In the left graph you can see how these effects are caused. The point in which both the aggregation rate and the depletion rate meet is the steady state since aggregation and depletion are equally large and thus dR/dt is zero. When changing the coefficients of each curve we observe the expected behavior. The steady state moves along both lines as the blue line changes due to the signal. This is the point I want to ask you if there are any questions concerning these principles. It is very important that you understand this very visualization in its entirety in order to be able to understand what is about to come. [answer questions] Now we will see, where to go from here. Biological example

Since this seminar also features biologists I want to present you another preliminary example that is actually taken from biology. And which process to choose but phosphorylization. The motor of our cells. What we observe in the wire diagram is the phosphorylization of our response protein such that is becomes RP. This uses up ATP and is controlled by a signal protein. Under the presence of water our phosphorylated protein looses the phosphor ion and goes back to its original state. If we put this into formulas we see that the rate of phosphorylation depends on the amount of unphorphorylated protein. We describe this with the difference of all protein that is present minus the phosphorylated amount. Please be careful as we are interested in the amount of phosphorylated protein RP not R. So all graphs that say R do in fact mean RP. On the left graph you can see the aggregation curve and the depletion curve respectively and the point where they meet, which is the steady state. Again we are given an ODE and can solve for R such that we get our steady state, which is plotted on the right hand side. If we compare this to the previous graph we see that the fact that the aggregation is not constant anymore, but depends on R as well gives us a hyperbolic shape. So, quick question to all biologists in the room, do you use this model for any calculations? Since obviously it is quite simplified. That is why we also have the more accurate model that is governed by Michaelis-Menten kinetics. It simply takes into account that rates can max out for processes no matter how strong the signal.

Buzzer

So now we are at the point where we can describe actual processes in biology. First of, we see that the differential equation is tweaked by a denominator that consits of the Michaelis factor which is constant and also the same variable as in the numerator. This causes the rates to take on a hyperbolic shape. Although this is no longer an ODE we can still solve it... or rather look up the solution on the internet or in a paper and we are given the solution below the equation. The so called 'Goldbeter-Koshland' function. You don't need to remember this equation exactly although we will use is wuite often. But you should remember that is has a sigmoidal shape similar to x/sqrt(1+x^2). But most important: sigmoidal shape! Again we can play around a bit and see what different coefficients cause on the plot. But let's look further into the shape of our steady state solution. As you may have noticed or already know, the sigmoid function if often used in machine learning for instance, since its shape is quite beneficial. For low signals we have a rather constant low response and for large signals we have a relatively constant large response. In between there is a steep ascent. Because this function is in its properties relatively similar to a button -like on a doorbell- that there is no response when the button is not pressed but a constant response if the button is pressed. Because of this, this network motif is also called a 'Buzzer'.

Detection of network motifs

Theoretically and Experimentally

Finding network motifs theoretically

Algorithm to detect network motifs:

  1. Create a network with same number of nodes as has the real network
  2. Assign the same number of incoming and outgoing edges to each node
    as has the corresponding node in the real network
  3. Scan network for all possible n-node subgraphs and record the number
    of occurrences of each subgraph
  4. Compare probabilities → cut-off probability

Study network motifs experimentally

  • Measure the transcriptional activity of promoters by means of reporter genes
  • Widely used: Green Fluorescent Protein (GFP)
  • Fast-folding GFP variants become fluorescent within minutes after transcription initiation
  • Promoter activation is detected by increased fluorescent signal
