Detection of network motifs

Theoretically and Experimentally

Finding network motifs theoretically

Algorithm to detect network motifs:

  1. Create a network with same number of nodes as has the real network
  2. Assign the same number of incoming and outgoing edges to each node
    as has the corresponding node in the real network
  3. Scan network for all possible n-node subgraphs and record the number
    of occurrences of each subgraph
  4. Compare probabilities → cut-off probability
